#  Creating Your First Project

Follow these steps to quickly set up a new project in the Its Magic game engine.

## 1. Starting a New Project

1.  In the Its Magic engine, select the **`New`** button.
    
     ![Image showing the template selection list.](/static/img/its-magic-fpi/project_creation.jpg)

2.  You have two main options for proceeding:

    * **Using a Template:** From the template list, select one of the available templates. Each template provides a sample game structure that helps you learn the basics of the Its Magic game engine. Select your desired template and click **Download**.
        
        ![Image showing the template selection list.](/static/img/its-magic-fpi/templates_list.jpg)

    * **Starting Fresh:** If you want a completely empty, fresh project without any starter assets, click on **`New Empty Project`**.

## 2. Naming and Creation

1.  In the next screen, type the **Project Name** for your new game.
2.  Select a location on your computer to save the project files.
3.  Click the **Confirm** button to generate the project files and open the Its Magic Editor.
     ![Image showing the template selection list.](/static/img/its-magic-fpi/empty_project.jpg)
